# Frontend для платформы квеста

Этот каталог содержит Vue-фронтенд для демоверсии веб-интерфейса конкурса.

## Установка

```bash
cd front
npm install
```

## Запуск

```bash
npm run dev
```

Фронтенд будет доступен по адресу `http://localhost:5173`.

## Настройка

Vite проксирует запросы `/api` на локальный backend `http://localhost`, поэтому перед запуском фронтенда нужно поднять backend.
