## 🚀 Установка и запуск

### Бэкенд (Laravel)

```bash
cd src

# Установка зависимостей
composer install

# Копирование .env файла
cp .env.example .env

# Генерация ключа приложения
php artisan key:generate

# Запуск миграций с seeders
php artisan migrate:fresh --seed

# Запуск сервера
php artisan serve
```

### Фронтенд (Vue 3)

```bash
cd front

# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```

## 📱 Структура приложения

### API Endpoints

#### Авторизация (Публичные)
```
POST   /api/login              - Вход
POST   /api/register           - Регистрация
```

#### Защищенные маршруты (требуют токен)
```
GET    /api/stages             - Список этапов
GET    /api/stages/{id}        - Детали этапа
GET    /api/stages/{id}/tasks  - Задачи этапа

GET    /api/teams              - Список команд
POST   /api/teams              - Создать команду
GET    /api/teams/{id}         - Детали команды
POST   /api/teams/{id}/join    - Присоединиться к команде
GET    /api/teams/{id}/members - Члены команды
GET    /api/teams/{id}/progress- Прогресс команды
GET    /api/teams/rankings     - Рейтинг всех команд

POST   /api/submissions        - Отправить решение задачи
GET    /api/users/{id}/progress- Прогресс пользователя
```

#### Админ маршруты
```
POST   /api/admin/stages       - Создать этап
PUT    /api/admin/stages/{id}  - Обновить этап
DELETE /api/admin/stages/{id}  - Удалить этап

POST   /api/admin/tasks        - Создать задачу
PUT    /api/admin/tasks/{id}   - Обновить задачу
DELETE /api/admin/tasks/{id}   - Удалить задачу
```

## 🔄 Workflow

1. **Вход в систему** - Пользователь входит с email и паролем
2. **Просмотр этапов** - Видит доступные этапы квеста
3. **Выполнение задач** - Может выполнять задачи текущего этапа
4. **Управление командами** - Может создавать и присоединяться к командам
5. **Просмотр прогресса** - Видит свой прогресс и прогресс команды
6. **Рейтинг** - Видит рейтинг команд по прогрессу

## 🚨 Возможные проблемы

Если при запуске возникают ошибки:

1. **401 Unauthorized** - Проверьте, что токен хранится в localStorage
2. **CORS ошибки** - Убедитесь, что бэкенд и фронтенд на одной сети или настроены правильно
3. **Миграции не запускаются** - Проверьте .env файл и подключение к БД
4. **npm install fails** - Удалите node_modules и package-lock.json, запустите заново
